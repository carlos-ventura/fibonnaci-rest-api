# Fibonacci REST JSON API

## Preconditions:

- Python 3

## Run local

### Install dependencies

```
pip install -r requirements.txt
```

### Run server

```
uvicorn app.main:app --reload
```

### Run test

```
pytest 
```


## API documentation (provided by Swagger UI)

```
http://127.0.0.1:8000/docs
```
